This API returns information about an area depending on what zip code it is in. It will return a JSON with the zip code, longitude and latitude, also the city and country name belonging to that zip code. This API is helpful for using autofill when someone inputs their zip code , or building a map based on zip codes. I used the Get option to request the JSON.
